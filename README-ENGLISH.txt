================================================
   Finance App - Quick Start Guide
================================================

Two ways to use the app:


METHOD 1: Expo Go (Fast & Easy)
================================

1. Install "Expo Go" from Google Play Store

2. Double-click: run-with-expo-go.bat

3. Scan QR Code with Expo Go

4. App runs on your phone!

Note: Phone and computer must be on same WiFi


METHOD 2: Build APK (Permanent Install)
========================================

Requirements:
- Java JDK 17: https://adoptium.net/
- Node.js: https://nodejs.org/

Steps:
1. Install Java JDK 17
2. Double-click: build-apk.bat
3. Wait 10-15 minutes
4. APK file will be at:
   android\app\build\outputs\apk\release\app-release.apk
5. Transfer to phone and install


APK Installation on Phone:
===========================

1. Copy app-release.apk to phone

2. Settings > Security > Unknown Sources
   Enable it

3. Click on APK file

4. Install


App Features:
=============

- Installment management
- Debt tracking
- Credit tracking  
- Check management
- Financial dashboard
- Reminders
- Persian calendar
- Fully offline
- Persian interface
